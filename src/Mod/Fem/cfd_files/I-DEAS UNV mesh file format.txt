I-DEAS UNV mesh file format

http://www.code-aster.org/forum2/viewtopic.php?id=14355

NODES
   in_file >> node_lab                // read the node label
                   >> exp_coord_sys_num       // (not supported yet)
                  >> disp_coord_sys_num      // (not supported yet)
                  >> color                   // (not supported yet)
                  >> xyz(0)                  // read x-coordinate
                  >> xyz(1)                  // read y-coordinate
                  >> xyz(2);                 // read z-coordinate


ELEMENTS
          in_file >> element_lab             // read element label
            >> f e_descriptor_id        // read FE descriptor id       
        >> phys_prop_tab_num       // (not supported yet)
              >> mat_prop_tab_num        // (not supported yet)
              >> color                   // (not supported yet)
              >> n_nodes;                // read number of nodes on element
              
              Apart form the id "2467" groups format(of elm or nodes) is Still Unknow

I take the following conlcusions

nodal groups:
node label/node id/unknow/unknow/node label/node id/unknow/unknow

elements group
element label/node id/unknow/unknow/element label/node id/unknow/unknow

in this case 7 means node and 8 means element (a triangle). Notice that there are 2 entities per row.
